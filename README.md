## 简介
这是一个IDEA的插件，用来记录开发过程中遇到的好的或者不理解的代码片段，方便快速查找复用。

## 使用方法
1. 选中想要记录的代码并右键选择添加词条菜单
2. 填写词条名称、解释、类型或修改代码内容并保存
3. 在IDEA底栏找到 Code Dictionary 查看、删除、编辑或保存本次记录的词条
4. 下次打IDEA目会自动加载上次保存的词条内容，方便查找使用
5. 如果移动了导出的文件，请重新导入

## 建议
阅读导出的MD文件推荐使用Typora软件
